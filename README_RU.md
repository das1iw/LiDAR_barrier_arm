# Обнаружение шлагбаума по данным LiDAR

## Язык
- Русский (этот файл)
- English → [README.md](README.md)

ROS2-пайплайн для обнаружения шлагбаума в реальном времени на основе LiDAR. 
Обрабатывает 3D облако точек из ROS2 bag-файла с использованием ROI-обрезки, 
статистического удаления выбросов, кластеризации DBSCAN и геометрии на основе PCA 
для определения состояния шлагбаума (OPEN/CLOSED) с частотой 10 Гц.

Датасет LiDAR доступен по ссылке:
https://drive.google.com/drive/folders/1hwPxpl8aj0GkSstgmdUSosJ_4X74jQD6?usp=share_link

Реализовано на Python (`alg_Python/`) и C++ (`alg_C++/`).

## ⚙️ Пайплайн обработки

1. ROI-обрезка – выделение области интереса, удаление земли  
2. Удаление выбросов – очистка шума в облаке точек  
3. Воксельная дискретизация – уменьшение плотности для ускорения  
4. Кластеризация DBSCAN – группировка точек в объекты  
5. Извлечение признаков (PCA) – вычисление длины, ширины и ориентации  
6. Выбор лучшего кластера – оценка по «тонкости» и длине  
7. Классификация – ОТКРЫТ/ЗАКРЫТ (OPEN/CLOSED) по порогу угла 80°  

## 📊 Производительность: C++ vs Python

Обе реализации дают идентичные результаты и достигают целевых 10 Гц, 
однако существенно различаются по скорости.

| Метрика            | C++              | Python |
|--------------------|------------------|--------|
| Загрузка данных    | 4 сек            | 102 сек|
| Время на кадр      | 17 мс            | 88 мс  |
| Частота (FPS)      | 10.00 Гц         | 9.98 Гц|
| Ускорение старта   | в 25 раз быстрее |    –   |
| Ускорение на кадр  | в 5 раз быстрее  |    –   |

Результаты алгоритма совпадают (один и тот же кластер, угол и состояние).

> C++ рекомендуется для продакшена и встраиваемых систем.  
> Python рекомендуется для прототипирования и разработки алгоритмов.
